# AssetOps Discovery

> Research repository for validating a SaaS opportunity in asset operations for field service and engineering companies.

---

# Цель проекта

Цель проекта — определить, существует ли достаточно большой рынок для SaaS, предназначенного для учета и управления дорогостоящими мобильными активами инженерных компаний.

Проект находится на стадии **Customer Discovery**.

Разработка продукта не начинается до подтверждения существования рынка и повторяемой бизнес-боли.

---

# Что исследуется

Проект исследует компании, в которых:

* дорогостоящее оборудование постоянно перемещается между объектами;
* существует несколько мобильных инженерных бригад;
* отсутствует прозрачный контроль местонахождения активов;
* потери оборудования приводят к финансовым потерям.

Основная задача — подтвердить существование повторяемой проблемы, а не разработать решение.
